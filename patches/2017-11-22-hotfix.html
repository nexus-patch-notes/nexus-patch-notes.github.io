Heroes of the Storm Patch Notes
2017-11-22 Hotfix Patch 2.29.1.59799

General

For general changes and/or gameplay updates, check out the official Patch Notes.

Heroes

Alexstrasza

Bug Fixes

  • Canceling Dragonqueen after entering or exiting the Haunted Mines will no longer prevent Alexstrasza from casting her other Abilities.

Brightwing

Bug Fixes

  • Soothing Mist will no longer passively Heal nearby allies while Brightwing is piloting a Vehicle.

Zul'jin

Bug Fixes

  • Completing the You Want Axe? Quest using the Wrong Place Wrong Time Talent will no longer cause Twin Cleave's visual effects to disappear during its second revolution.
  • Fixed an issue causing the Movement Speed bonus granted by Eye of Zul'jin to be lost on death.
  • Eye of Zul'jin's Talent description will no longer incorrectly append to the Grievous Throw tooltip.
  • No Mercy's Talent description will now correctly append to the Grievous Throw tooltip.
  • Recklessness is no longer incorrectly categorized as a passive Talent in the Talent Pane.
  • Voodoo Shuffle's tooltip now correctly lists a 40% reduction in Regeneration's Mana cost and cooldown, rather than 33%.
  • Voodoo Shuffle's Talent description will now correctly append to the Regeneration tooltip.
  • Buff icons in Zul'jin's Buff Bar will now match the order in which their associated Talents were selected.
